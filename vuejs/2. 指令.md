### v-bind
#### 参数
- HTML 属性名称
``` HTML
<!--在这里 href 是参数，告知 v-bind 指令将该元素的 href 属性与表达式 url 的值绑定。-->
<a v-bind:href="url"></a>
```

### v-if
> 如果 v-if 指令绑定的表达式的值为真，则显示此 DOM 结构。
``` HTML
<!-- Handlebars 模板 -->
{{#if ok}}
  <h1>Yes</h1>
{{/if}}

<!-- 使用 v-if 指令实现同样的功能 -->
<h1 v-if="ok">Yes</h1>
```

### v-model

### v-on
> 绑定一个事件监听器，通过它调用 Vue 实例中定义的方法。

### v-once
> 执行一次性地插值，当数据改变时，插值处的内容不会更新。
