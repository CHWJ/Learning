## 基本类型
- 整型
    - `byte` (1个字节)
    - `short`(2个字节)
    - `int`  (4个字节)
    - `long` (8个字节)
    - `BigInteger`(8+个字节)


### 整型
### int
#### 极值
- Integer.MIN_VALUE 最小整数，为`0x80000000`
- Integer.MAX_VALUE 最大整数，为`0x7fffffff`
### long
- 带 `L` 后缀

## 进制
- 二进制 `0b` 后缀。示例：0b1001(二进制) = 9(十进制)
- 八进制 `0` 后缀。示例：011(八进制) = 9(十进制)
- 十进制 
- 十六进制 `0x` 后缀
