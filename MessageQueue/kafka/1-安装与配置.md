### 准备

- 安装jdk8，并设置系统环境变量 JAVA_HOME

### 下载

- [下载二进制包](https://kafka.apache.org/community/downloads/)
- 解压到某个盘符的根目录下，且路径不要包含空格

### 配置

- config\zookeeper.properties
  ``` ini
    # 建议新建一个专门的 data 文件夹
    dataDir=C:/kafka/data/zookeeper
  ```
- config\server.properties
  ``` ini
    # 修改日志/数据存放路径
    log.dirs=C:/kafka/data/kafka
    # 如果需要外网或局域网访问，取消注释并修改为当前服务器的 IP
    # listeners=PLAINTEXT://your_server_ip:9092
  ```
